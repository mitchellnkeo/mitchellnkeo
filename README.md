Hi 👋 My name is Mitchell Keo
=============================

Full Stack Engineer
-------------------

I have just recently left active duty military and am currently a veteran looking to break into the software engineering field. I just completed a 17 week Software Engineering Immersive with Galvanize: Operation Level Up, a full stack development program geared toward transitioning military members. I chose software engineering because I wanted to be in a career field that values creative problem solving, collaborative teamwork, and innovation - and I hope to bring these values to a team that shares those ideals as well.

* 🌍  I'm based in Seattle
* ✉️  You can contact me at [mitchellnkeo@gmail.com](mailto:mitchellnkeo@gmail.com)
